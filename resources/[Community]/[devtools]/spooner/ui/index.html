Menus

Spawn menu - F

The Spawn fournit des listes consultables pour sélectionner une entité à générer. Un clic gauche sur une entité la définit comme votre spawn actuel.

Si une entité n’est pas incluse dans la liste, vous pouvez toujours la générer en entrant le nom complet du modèle dans le champ de recherche et en cliquant sur Spawn By Name.

Un clic droit sur une entité dans l’un des menus spawn ajoutera cette entité en tant que favori. Cliquer sur le bouton Favoris basculera en affichant uniquement vos entités préférées.

Database menu - X

The Database stocke une liste d’entités. Lorsqu’une entité est générée, elle est automatiquement ajoutée à la base de données active. Les entités existantes peuvent être ajoutées/supprimées de la base de données via le Properties menu.

  • Cliquez avec le bouton gauche de la souris sur une entité pour l’ouvrir dans le Properties menu
  • Cliquez avec le bouton droit de la souris sur une entité pour la supprimer
  • Click Delete All pour supprimer toutes les entités de la base de données

Properties menu - e

The Properties le menu répertorie et vous permet de modifier les propriétés d’une entité.

Les propriétés étiquetées comme (locales) ne sont pas synchronisées sur le réseau, de sorte que les modifications apportées à ces propriétés ne seront visibles que par vous.

Save/Load Database menu - J

The Save/Load Database vous permet de stocker votre base de données actuelle avec un nom, puis de charger à nouveau toutes les entités à partir de celle-ci ultérieurement.

  • Pour enregistrer votre base de données actuelle, entrez un nom dans le champ et cliquez sur Save.
  • Pour charger une base de données enregistrée, cliquez avec le bouton gauche de la souris sur le nom de la base de données.
  • Pour supprimer une base de données enregistrée, cliquez avec le bouton droit sur le nom de la base de données.
  • Pour importer une base de données ou exporter la base de données actuelle, cliquez sur Import/Export.

Vérification de l' Charge par rapport à la position du curseur apparaîtra les entités dans la base de données sélectionnée par rapport à la position actuelle du curseur, plutôt que par rapport à l’endroit exact où elles ont été placées à l’origine.

Vérification de l' Remplacer la base de données actuelle remplacera votre base de données actuelle par la base de données chargée, plutôt que de fusionner les deux.

Vérification de l' Save/Load deletions enregistre les entités que vous supprimez et les supprime à nouveau lorsque la base de données est chargée.

Import/Export menu

The Import/Export menu vous permet d’importer et d’exporter des bases de données dans un certain nombre de formats différents:

Format Description Export? Import?
Spooner DB JSON The native format used by the spooner Yes Yes
Map Editor XML XML format used by the Lambdarevolution map editor and the objectloader resource Yes No
Ymap Native map format used by GTA V/RDR2 Yes Yes
propplacer JSON RedEM:RP propplacer JSON database Yes No
Spooner Backup Backup of all spooner databases Yes Yes

Pour exporter, sélectionnez le format souhaité et cliquez sur Export. La sortie sera affichée dans la zone de texte et vous pourrez la copier pour l’enregistrer dans un fichier externe.

Pour importer, collez l’entrée dans la zone de texte, sélectionnez le format approprié, puis cliquez sur Import. Les objets importés seront ajoutés à votre base de données actuelle.

Entering a URL of a JSON/XML file in the Import from URL field and clicking Import vous permet d’importer à partir de sources Web externes, telles que pastebin.com, sans avoir besoin de copier et coller. Assurez-vous que l’URL pointe vers la version brute du fichier lors de l’utilisation de ces services.
