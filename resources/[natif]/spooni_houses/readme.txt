The first Spooni House package is finally ready.

How is it installed?

Go on your Email Account from Tebex. You get a Email for the Download Link.

There aree the Files you need.

Copy spooni_houses into the resources folder. 

Then in the server.cfg
write

ensure spooni_houses

Copy the objects.lua file to the Spooner folder

resources/spooner/data/rdr3/ there.

feel free to overwrite the file, all Spooner objects are in the list.

then restart server. 

Start Spooner under Objects, the list of Spooni Houses and Props is now at the top. 

It is important that all objects are in it, but of course only those that you have purchased will work.
